Instalación de Django

Guía paso a paso para configurar Django con tu template BizLand

1

Crear entorno virtual

Un entorno virtual aísla las dependencias de tu proyecto.

python -m venv myenv
# Windows
myenv\Scripts\activate

# macOS/Linux
source myenv/bin/activate
2

Instalar Django

Instala Django usando pip dentro de tu entorno virtual.

pip install django
Nota: También puedes instalar una versión específica con pip install django==4.2.3
3

Crear proyecto Django

Inicia un nuevo proyecto Django.

django-admin startproject myproject .
Nota: El punto al final crea el proyecto en el directorio actual en lugar de crear una subcarpeta.
4

Crear una aplicación

Crea una aplicación dentro de tu proyecto.

python manage.py startapp myapp
5

Estructura de carpetas

Crea manualmente las carpetas necesarias para tu template.

myproject/
  templates/
  static/
    css/
    js/
    images/
  manage.py
  myapp/
  myproject/
6

Configurar settings.py

Edita el archivo de configuración para incluir tu app y configurar los directorios.

# Agregar tu aplicación
INSTALLED_APPS = [
  'myapp',
]

# Configurar templates
import os
TEMPLATES = [
  {
    'DIRS': [os.path.join(BASE_DIR, 'templates')],
  }
]

# Configurar archivos estáticos
STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]
7

Ejecutar migraciones

Configura la base de datos inicial de Django.

python manage.py migrate
8

Ejecutar el servidor

Inicia el servidor de desarrollo de Django.

python manage.py runserver
Nota: Ahora puedes acceder a tu proyecto en http://localhost:8000

¿Necesitas más ayuda?

Visita la documentación oficial de Django para más información.
